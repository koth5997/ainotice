# 🚀 AI 전산 공지 생성기 - 빠른 설치 가이드

## ⚡ 5분 안에 시작하기

### 1단계: Python 설치 확인

명령 프롬프트(cmd) 또는 PowerShell을 열고:

```bash
python --version
```

Python 3.10 이상이어야 합니다. 없다면 [python.org](https://www.python.org/downloads/)에서 설치하세요.

### 2단계: 프로젝트 폴더 열기

1. `ai-notice-generator` 폴더를 원하는 위치에 저장
2. 해당 폴더에서 우클릭 > "터미널에서 열기" 또는 명령 프롬프트에서:

```bash
cd 폴더경로\ai-notice-generator
```

### 3단계: 가상환경 설정

```bash
# 가상환경 생성
python -m venv venv

# 가상환경 활성화
# Windows:
venv\Scripts\activate

# macOS/Linux:
source venv/bin/activate
```

### 4단계: 패키지 설치

```bash
pip install -r requirements.txt
```

⏱️ 약 1-2분 소요

### 5단계: Google Gemini API 키 발급

1. 브라우저에서 접속: https://makersuite.google.com/app/apikey
2. Google 계정으로 로그인
3. "Create API Key" 클릭
4. API 키 복사 (예: AIzaSy...)

### 6단계: 환경 설정

1. `.env.example` 파일을 `.env`로 이름 변경 (또는 복사)
2. `.env` 파일을 메모장으로 열기
3. API 키 입력:

```env
GEMINI_API_KEY=여기에_복사한_API_키_붙여넣기

HOST=127.0.0.1
PORT=8000
```

4. 저장 후 닫기

### 7단계: 실행!

```bash
python main.py
```

다음과 같은 메시지가 나타나면 성공:

```
============================================================
🚀 AI 전산 공지 생성기 시작 (채팅 모드)
============================================================
📍 채팅 페이지: http://127.0.0.1:8000
📝 공지 관리: http://127.0.0.1:8000/notices
============================================================
```

### 8단계: 웹 브라우저 열기

브라우저에서 `http://127.0.0.1:8000` 접속

## ❓ 자주 묻는 질문

### Q: 가상환경이 활성화되지 않아요 (Windows)

**A:** PowerShell에서 다음 명령어 실행:

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

그 후 다시 `venv\Scripts\activate` 실행

### Q: 포트가 이미 사용 중이라고 나와요

**A:** `.env` 파일에서 다른 포트 번호로 변경:

```env
PORT=8080
```

### Q: API 키 오류가 나요

**A:** 확인사항:
1. `.env` 파일이 프로젝트 폴더에 있는지
2. API 키가 정확히 복사되었는지 (공백 없이)
3. Google AI Studio에서 API 키가 활성화되었는지

### Q: 가상환경을 종료하려면?

**A:** 터미널에서:

```bash
deactivate
```
